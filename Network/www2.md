# 물리계층
----------------
#### 물리계층이란?
```
디지털 신호전달

실제 장치들을 연결하기 위해 필요한장치(연결을하기위한 물리적장치)

대표적으로 케이블/리피터/허브
```
#### 케이블(CABLE)
```
전기선
전기를 이용해서 신호를 전달하는 선
```
